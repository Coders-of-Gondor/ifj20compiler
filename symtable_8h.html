symtable_find_symbol looks for a symbol in symbol table matching a key

key is used to search for an entry in a symtable. If key matches an entry in a symtable, the returned pointer aims at the data. If not, the returned pointer is NULL;

symtable_stack_pop removes the topmost symtable from the stack

symtable_stack_pop pops a stack only if there's more than one symtable in the stack. The popped symtable is not kept in memory.
